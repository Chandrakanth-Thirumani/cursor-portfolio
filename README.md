# cursor-portfolio

This repository documents my setup of Cursor IDE and related AI coding add-ons as part of a development environment configuration assignment.

**All assignment steps (1–7) are completed.**

## Tools Installed

| Tool | Description |
|------|-------------|
| [Cursor IDE](https://cursor.com/) | AI-powered code editor built on VS Code, used as the primary development environment |
| **Claude Code** (Cursor extension) | Anthropic's Claude integration for in-editor AI coding assistance |
| **Codex** (Cursor extension) | OpenAI's Codex integration for in-editor AI coding assistance |

## Steps Completed

1. **Installed Cursor IDE** — Downloaded and installed from [cursor.com](https://cursor.com/).

2. **Added the Claude Code add-on** — Opened Cursor → Extensions → searched for "Claude Code" → installed the extension. *(See [Issues](#issues) below regarding sign-in.)*

3. **Added the Codex add-on** — Opened Cursor → Extensions → searched for "Codex" → installed the extension and signed in successfully.

4. **Created a public GitHub repository** — Created [cursor-portfolio](https://github.com/Chandrakanth-Thirumani/cursor-portfolio) on GitHub.

5. **Opened the repository in Cursor** — Cloned the repo locally and opened it in Cursor IDE.

6. **Created this README.md** — Documented the tools, steps, and issues encountered during setup.

7. **Committed and pushed to GitHub** — Changes committed to the `main` branch and pushed to the remote repository.

## Issues and How They Were Solved

### Claude Code sign-in requires a Pro or Max account

**Issue:** After installing the Claude Code extension, sign-in failed because Claude Code requires an Anthropic **Pro** or **Max** subscription. I do not currently have either plan.

**Impact:** The extension is installed, but full Claude Code functionality is unavailable until a qualifying Anthropic subscription is active.

**Resolution / workaround:**
- Installed the extension as required by the assignment.
- Attempted sign-in and confirmed the subscription requirement.
- Continued with **Codex**, which signed in successfully and provides in-editor AI assistance for development work in the meantime.
- Will sign in to Claude Code once a Pro or Max subscription is available.
